# KaTex

相似庫

- Tex (1978)
- LaTex (1984)
- KaTex 很方便與瀏覽器做結合, 提供相關的js庫
  - [Github](https://github.com/KaTeX/KaTeX)
  - [wiki](https://en.wikipedia.org/wiki/KaTeX)


如果要在md中使用，又不想額外寫或引用其他套件，可以靠一些網站的API來幫忙，如下

```
![](http://latex.codecogs.com/gif.latex?\\frac{1}{1+sin(x)})
```

結果如下:
![](http://latex.codecogs.com/gif.latex?\\frac{1}{1+sin(x)})
